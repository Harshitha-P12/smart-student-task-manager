📚 Smart Student Task Manager

A modern and user-friendly task management web application designed specifically for students to organize their academic life efficiently.

🚀 Features:
- 🔐 User Authentication (Login/Register/Forgot Password)
- ✅ Add, Update, and Delete Tasks
- 📅 Calendar View for Task Planning
- 🔥 Daily Streak Tracking System
- 📊 Task Count Dashboard
- 💻 Clean and Responsive UI

🛠️ Tech Stack:
- Frontend: HTML, CSS, Bootstrap
- Backend: Flask (Python)
- Database: SQLite / MySQL

🎯 Purpose:
This project helps students stay organized, improve productivity, and maintain consistency in completing their daily tasks.

📌 Future Enhancements:
- Notifications & Reminders
- AI-based task suggestions
- Mobile responsiveness improvements
